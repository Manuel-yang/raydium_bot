# robot

My cool TypeScript app

## Quick Start

Get started developing...

```shell
# install deps
npm install

# run in development mode
npm run dev

# run tests
npm run test
```

## Env
```shell
cp .env.example .env
```
set your .env config 

## Run with pm2

install pm2 in vps

run without compile
```shell
pm2 start "npm run dev" --name raydiumTracker
```
